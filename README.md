# Job Recommender

Консольный симулятор системы рекомендаций вакансий. Позволяет создавать профили пользователей и вакансии, а затем подбирать наиболее подходящие вакансии для каждого пользователя.

## Запуск

Приложение можно запустить нажатием кнопки "Run" в IntellijIdea, либо командой:
```bash
./gradlew run
```

## Команды

| Команда | Описание |
|---|---|
| `user <name> --skills=... --exp=...` | Добавить пользователя |
| `user-list` | Вывести список пользователей |
| `job <title> --company=... --tags=... --exp=...` | Добавить вакансию |
| `job-list` | Вывести список вакансий |
| `suggest <username>` | Показать подходящие вакансии для пользователя |
| `exit` | Завершить работу |

## Примеры

### Добавление пользователей
```
user alice --skills=java,ml,linux --exp=2
user bob --exp=10 --skills=java,python
```

### Добавление вакансий
```
job Backend_Dev --company=VK --tags=java,backend,linux --exp=1
job ML_Engineer --tags=ml,python --company=Google --exp=3
```

### Просмотр списков
```
user-list
alice java,linux,ml 2
bob java,python 10

job-list
Backend_Dev at VK
ML_Engineer at Google
```

### Рекомендации вакансий
```
suggest alice
Backend_Dev at VK
ML_Engineer at Google
```

## Логика подбора

Релевантность вакансии вычисляется как количество совпадений между навыками пользователя (`skills`) и тегами вакансии (`tags`). Если опыт пользователя меньше требуемого — результат делится пополам. Выводится не более 2 наиболее подходящих вакансий.

## Замечания

- Повторное добавление пользователя или вакансии с тем же именем игнорируется
- Данные хранятся в памяти приложения и не сохраняются при его остановке
